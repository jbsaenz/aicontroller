# Project Assumptions

1. Synthetic data is acceptable because assignment explicitly allows a synthetic dataset.
2. Telemetry cadence is fixed and regular (planned: 10-minute intervals).
3. Miner fleet behavior is mode-dependent (`eco`, `normal`, `turbo`).
4. Thermal, electrical, and performance stress jointly influence degradation/failure probability.
5. Labels represent a risk horizon, not necessarily immediate hard shutdown.
6. Prototype supports decision assistance, not autonomous hardware control.
